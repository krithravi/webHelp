Mapping Keys

The Problem

The Caps Lock key serves mainly as a source of annoyance, and no one really uses it. However, the Esc key is quite useful (I'm primarily a Vim user), but is far too hard to reach. Currently, Éowyn has altered key settings to make life a bit eaiser. The Caps Lock key now acts like the Esc key.

Stuff that didn't work

I first tried this using the command xmodmap -e `clear Lock' -e `keycode 0x42 = Escape' but for some reason, it stopped working for one of my TEX  files.
